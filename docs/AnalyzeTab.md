# AnalysisTab Component Documentation

Maps sections in `components/report/AnalysisTab.tsx` to AI prompt data structures.

## Data Structure

The AnalysisTab displays `report_part_2` generated by Gemini AI:

```
MarketingReport
├── report_part_1: Channel info and video data
├── report_part_2: Strategy analysis (displayed in AnalysisTab)
└── report_part_3: Insights and action plans (displayed in EvaluationTab)
```

## Section Mapping

| Section | Prompt Path | Description |
|---------|-------------|-------------|
| Brand Identity | `strategy_analysis.brand_identity` | Visual style, tone, positioning |
| Ad Strategy | `ad_strategy` | Overview, angles, target audience |
| Content Pillars | `strategy_analysis.content_pillars[]` | Purpose and description per pillar |
| Content Niche | `strategy_analysis.content_niche_analysis` | Primary/sub niches, categories |
| Content Structure | `strategy_analysis.content_structure_analysis` | Hooks, storytelling, CTA |
| Funnel Analysis | `funnel_analysis` | TOFU/MOFU/BOFU breakdown |
| Audience Analysis | `audience_analysis` | Demographics, behavior, psychographics |
| Audience Personas | `audience_personas[]` | Min 2 personas with details |
| Content Calendar | `content_calendar` | Best days/times, frequency, content mix |
| SEO Analysis | `seo_analysis` | Keywords, tags, optimization |
| Growth Opportunities | `growth_opportunities[]` | Priority-ranked opportunities |
| Quantitative Synthesis | `quantitative_synthesis` | Stats, health metrics, performance |

## Data Sources

| Type | Examples |
|------|----------|
| **Real Data** | Video count, views, likes, comments, publish times, tags |
| **Calculated** | Engagement rate, posting frequency, peak times |
| **AI Estimated** | Demographics, personas, content structure, competitors |

## Key Files

- **Component**: `components/report/AnalysisTab.tsx`
- **Prompt**: `lib/prompts/marketing-report.ts`
- **Types**: `lib/types.ts`
- **Gemini**: `lib/gemini.ts`

## Limitations

1. AI cannot watch videos - analysis based on metadata only
2. Demographics are AI estimates (YouTube API doesn't provide this)
3. Competitor analysis is AI speculation (no actual data fetched)
