## üêõ Known Issues & Limitations

### Current Limitations
1. **No Video Viewing**
   - AI cannot watch actual videos
   - Analysis based on titles, descriptions, tags only
   - Content structure insights are metadata-based

2. **Demographic Data Accuracy**
   - Demographics are AI estimates
   - YouTube API doesn't provide actual audience data
   - Based on content language, topic, and style

3. **Competitor Analysis**
   - Currently speculative by AI
   - No real competitor data fetched
   - Recommendations are generalized

4. **Video Limit**
   - Maximum 50 videos per analysis
   - Filtered to last 30 days (or top 10 if none)
   - Long-running channels may lack historical context

5. **API Quota Constraints**
   - YouTube API: 10,000 units/day (free tier)
   - Gemini API: Rate limits apply
   - May require user API keys for heavy usage

### Planned Fixes
- [ ] Add video viewing capability (future API integration)
- [ ] Implement YouTube Analytics API (requires channel owner auth)
- [ ] Add historical data storage
- [ ] Implement API key rotation
- [ ] Add quota usage dashboard
